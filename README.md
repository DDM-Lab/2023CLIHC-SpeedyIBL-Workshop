# 2023CLIHC-SpeedyIBL-Workshop

This repo contains the materials for the workshop at CLIHC 2023, titled "Applications of Instance-Based Learning Theory: Using the SpeedyIBL Library to Construct Computational Models" by Erin H. Bugbee, Thuy Ngoc Nguyen, and Cleotilde Gonzalez.

The slides for the workshop can be found [here](https://erinbugbee.quarto.pub/applications-of-instance-based-learning-theory-using-the-speedyibl-library-to-construct-computational-models/#/title-slide).
